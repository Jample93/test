##### 강조

*내가 웃는게 웃는게 아니야*  
_내가 웃는게 웃는게 아니야_  
  
**내가 웃는게 웃는게 아니야**  
__내가 웃는게 웃는게 아니야__    
  
*내가 웃는게 **웃는게** 아니야*  
방법: * 혹은 _ 표시를 사용, 강하게 강조하고 싶을 땐 ** 혹은 __
